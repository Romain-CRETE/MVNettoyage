Description :
MVNettoyage est un site web réalisé pour un professionnel afin de présenter les services d’une entreprise de nettoyage.
Ce projet a été développé dans le cadre de ma reconversion vers le développement web, avec une immersion pratique me permettant de travailler sur un projet réel.

Le code a été réalisé sur VS Code . FileZilla a été utilisé pour la mise en ligne.
Le site est hébergé sur OVH et est pleinement fonctionnel en ligne. 

Objectifs et apprentissages :

Développer un site web fonctionnel pour un client réel.

Appliquer JavaScript pour des fonctionnalités interactives.

Mettre en ligne un site web sur un serveur professionnel (OVH).

Travailler avec rigueur et organisation dans un contexte  réel.

Outils utilisés :

VS Code
FileZilla
OVH pour l’hébergement
